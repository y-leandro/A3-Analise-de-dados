A fonte dos dados utilizados na pesquisa vem de arquivos "PGN", sigla para "Portable Game Notation" (Traduz-se para "Notação de Jogo Portátil"). O PGN é um formato de texto responsável por documentar jogos de xadrez, tanto sobre seus lances quanto os dados da partida, envolvendo informações sobre os jogadores, data da partida jogada, torneio do jogo (caso a partida faça parte de um), lado vencedor, entre outros.
