
## Análise de Dados sobre Xadrez.

O xadrez é um dos jogos mais estudados da história, milhares de partidas são digitalizadas por meio do PGN, um padrão textual que registra lances, tempo de jogo, rating dos jogadores e resultado, tudo em estrutura padronizada e de fácil processamento automatizado que abre espaço para análises e perguntas além da tradicional análise tática. Com a análise de dados é possível identificar padrões de vitória associados a aberturas específicas, se a idade do jogador interfere no desempenho, dentre outras perguntas. Neste presente trabalho aplicamos métodos estatísticos e ferramentas de ciência de dados a partidas de xadrez com o objetivo de explorar as bases de dados para formalizar novos conteúdos e ensaios sobre o mundo do xadrez profissional.
